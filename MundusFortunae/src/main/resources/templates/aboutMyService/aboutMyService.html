실시간 플레이와 가벼운 운빨(?)의 재미가
만나는 소셜 미니게임 월드

Mundus Fortunae는 실시간 게임과 채팅 기반 소셜 경험을 중심으로 설계된 미니게임 플랫폼입니다.

방 생성 → 참여자 입장 → 진행 상황 브로드캐스트 → 채팅과 로그 공유까지, 한 번의 흐름으로 매끄럽게 이어지는 플레이 경험을 목표로 합니다. 단순한 규칙 속에서도 ‘함께’일 때 더 재밌는 순간을 만들고자 했습니다.

🎲

실시간 미니게임

주사위/턴 기반 게임 등 빠르게 즐길 수 있는 룰 중심의 라이트한 게임 구조를 지향합니다.

👥

방 & 참여자 시스템

로비에서 방을 탐색하고, 참여자 목록과 상태를 실시간으로 확인하며 자연스러운 멀티플레이 흐름을 제공합니다.

💬

채팅 & 게임 로그

채팅과 시스템 로그를 분리해 플레이 정보와 대화를 동시에 놓치지 않도록 구성합니다.

🔐

신뢰 가능한 인증 구조

JWT 기반 인증 흐름을 고려해 게임 접근과 사용자 경험을 안정적으로 연결하는 방향을 추구합니다.

기술 기반

Mundus Fortunae는 Spring MVC Thymeleaf WebSocket PostgreSQL JWT 등 실무 친화적인 구성을 바탕으로 확장 가능한 실시간 서비스 구조를 실험하고 있습니다.

운과 전략 사이, 그리고 사람과 사람 사이에서 생기는 재밌는 순간들.
